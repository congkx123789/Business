## 💡 Best Practices khi Code

### 1. **State Management (Rule #5):**
```typescript
// ✅ ĐÚNG: Server data → React Query
const { data: stories } = useQuery(['stories'], () => api.getStories())

// ✅ ĐÚNG: UI state → Zustand
const { isSidebarOpen, toggleSidebar } = useUIStore()

// ❌ SAI: Không dùng useState cho server data
const [stories, setStories] = useState([]) // ❌
```

### 2. **Component Structure:**
```
components/
├── ui/              # Atoms (shadcn/ui) - Reusable primitives
├── shared/          # Organisms - Reusable complex components
└── features/        # Templates - Feature-specific components
    └── Reader/
        ├── ReaderContainer.tsx    # Main container
        ├── ReaderContent.tsx      # Content display
        └── ReaderControls.tsx      # Controls bar
```

### 3. **API Calls:**
```typescript
// lib/api.ts
export const api = {
  getStories: () => axios.get('/api/stories'),  // Calls 1-gateway
  getChapter: (id: string) => axios.get(`/api/chapters/${id}`),
}

// lib/hooks/useStories.ts
export const useStories = () => {
  return useQuery(['stories'], api.getStories)
}
```

### 4. **Import từ 7-shared:**
```typescript
// ✅ Import types/DTOs từ shared package
import { StoryDto, ChapterDto, UserDto } from '@shared/types'
import { ReadingPreferences } from '@shared/models'
```

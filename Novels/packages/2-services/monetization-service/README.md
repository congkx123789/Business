# Monetization Service

Monetization Service - PPC, Virtual Currency, Privilege, Subscriptions

## Port
- HTTP: 3010
- gRPC: 50060

## Modules
- **Wallet**: Virtual Currency System
- **PPC**: Pay-Per-Chapter Business Model
- **Privilege**: Advanced Chapters (book-specific, monthly reset)

## Database
- SQL Server (same as other services)

## Contribution Guide
- Create a file within 48h of incident close
- Include clear timeline, impact, root causes, CPAs with owners/dates
- Tag with affected SLO(s) and budget impact

## Review Cadence
- Weekly: skim new PMs in on-call review
- Monthly: trends and repeated contributors
- Quarterly: feed into SLO review decisions

# SRE Governance Loop

Measure → Act → Respond → Learn (repeat)

- Measure: SLOs/SLIs, dashboards, burn alerts
- Act: reliability work from reviews and postmortems
- Respond: on-call policy, incident playbook, Alertmanager routes
- Learn: blameless postmortems, quarterly reviews

Operational cadence:
- Weekly: error-budget review (issue template: Weekly Error-Budget Review)
- Quarterly: SLO review (issue template: Quarterly SLO Review)

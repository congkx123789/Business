# Baseline Security Code Reviews

Scope (one-time deep dive):
- Payment flows
- User profile flows
- Orders flows

Deliverables:
- Use the checklist to review the listed areas
- Create a report per area in `security/reviews/reports/`
- Open and merge all must-fix issues

How to:
1) Copy the checklist to a new file under `reports/AREA-YYYYMMDD.md`.
2) Walk the code paths end-to-end and record findings.
3) File tickets and link them in the report.
4) Re-test after fixes are merged.

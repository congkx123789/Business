# OWASP Top 10 Training

Objective: All developers complete OWASP Top 10 training. Attendance must reach 100% (PCI DSS 6.x).

Artifacts:
- Slides: Create or link to course material (company LMS or OWASP resources).
- Roster: Use `roster-template.csv` to track attendance.

Suggested agenda (60–90 minutes):
1) Injection & XSS
2) Broken Authentication & Access Control
3) Cryptographic Failures & Secrets Handling
4) Security Misconfiguration
5) Vulnerable & Outdated Components (SCA)
6) Logging/Monitoring & Incident Response

Completion:
- Record attendee name, team, date completed, and verifier.
